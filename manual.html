Manual - Sistema de Poltronas Inteligentes

Este guia completo foi criado para ajudar proprietários de cinemas a configurar o sistema LYS de forma fácil e segura. Nosso objetivo é tornar a instalação e a utilização do sistema o mais simples possível, garantindo que sua equipe tenha acesso rápido às funcionalidades essenciais para uma operação eficiente.
  • Instalação
  • Configuração
  • Cuidados
  • Suporte

Instalação

Instruções detalhadas sobre como instalar o sistema de poltronas inteligentes. Inclua informações sobre como conectar os LEDs, configurar o QR code e ajustar as configurações iniciais.

Configuração

Explique como configurar o sistema, como personalizar as luzes de LED, ajustar as preferências do cliente e usar a interface de funcionários.

Cuidados

Orientações sobre como cuidar do sistema de poltronas inteligentes, garantir a durabilidade dos LEDs e realizar a manutenção periódica.

Suporte

Informações sobre como entrar em contato com a equipe de suporte, como resolver problemas comuns e onde encontrar mais recursos ou manuais.
